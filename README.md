## Features
- Always has acces to the latest movies.
- This is because the API will always display the latest movies.
- Images along side the title of the movie presented.
- Backend REST api that is able to acces all reviews. 
- Reviews are stores in a MongoDB database.
- Implements a DAO pattern for the backend.
